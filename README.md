# [PhoNER_COVID19-Replication]

An unofficial implementation and replication of the paper: **"PhoNER_COVID19: Vietnamese Named Entity Recognition Dataset for COVID-19"**.

---

## 📌 Introduction

This repository contains the source code and experimental setups for replicating the research paper **"PhoNER_COVID19: Vietnamese Named Entity Recognition Dataset for COVID-19"**. This project is conducted as part of the Data Mining course at the **University of Information Technology (UIT)**.

The primary objective of this study is to reproduce the empirical results presented by the original authors, specifically focusing on named entity recognition (NER) for Vietnamese health-related texts using state-of-the-art language models like **PhoBERT** and deep learning architectures such as **BiLSTM-CNN-CRF**. Through this replication, we aim to validate the model's performance, evaluate its training stability on Kaggle environments, and analyze the specific challenges of tokenization and entity boundary detection in the Vietnamese medical domain.
